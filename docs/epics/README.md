# Epics

One file per epic, named `E<n>-<slug>.md`: the epic's outcome line, its link into the PRD section that specifies it, and its stories with their tick state. Ticks live here — nowhere else — so a tick's blast radius is one epic's file.

The epic order, one-line outcomes, and the MVP cut live in [`../../ROADMAP.md`](../../ROADMAP.md) — the index, deliberately kept free of story detail. Completed epics get a dated, append-only narrative in [`../write-ups/`](../write-ups/).

This directory is intentionally not listed here: the roadmap is the single index, and a second list would be one more thing to keep in sync.
